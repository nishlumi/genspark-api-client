# Genspark API クライアント デモアプリ

このアプリは、Genspark API Client ライブラリの機能を体験するためのシンプルな Web プレイグラウンドです。Web検索、画像検索、Webページ取得（クローラ）、ページ要約、および AI 画像生成のデモを実装しています。

## 主な機能
- **Web検索**: ウェブ上の情報を検索し、カード形式で結果を表示します。
- **画像検索**: 画像を検索し、詳細情報とともにリスト形式で表示します。
- **Crawler**: ウェブページの内容を取得し、Markdown 形式でレンダリングします。
- **Webページ要約**: 指定したURLの内容から、質問に対する回答を要約して取得します。
- **画像生成**: さまざまなAIモデルを使用して、プロンプトから高品質な画像を生成します。

## クイックスタート

### 1. 事前準備
- Node.js (LTS推奨)

### 2. インストール
`demo` ディレクトリに移動し、依存パッケージをインストールします。
```bash
cd demo
npm install
```

### 3. APIキーの設定
本アプリはセキュリティのため、APIキーをサーバー側で管理します。以下のいずれかの方法で設定してください。

- **環境変数に設定**:
  ```bash
  export GSK_API_KEY=あなたのAPIキー
  ```
- **.env ファイルを作成**:
  `demo` ディレクトリに `.env` という名前のファイルを作成し、以下を記述します。
  ```env
  GSK_API_KEY=あなたのAPIキー
  ```

### 4. アプリの起動
プロキシサーバーを起動します。
```bash
node server.js
```
または、
```bash
npm start
```
その後、ブラウザで以下のURLにアクセスしてください。
`http://localhost:3000`
* ポートは `npm start` 実行時の環境変数 `DEVPORT` で任意に変更可能です。

## プロジェクト構成
- `server.js`: CORSプロキシとして動作し、APIリクエストを処理する Express サーバー。
- `index.html`: フロントエンドのUI。
- `../genspark_api.js`: サーバーで使用されるコア API クライアントライブラリ。
